How-to
=
## XBee on Hana
Set to run in Coordinator and API mode, and open term.py for packet monitoring
- AP=1 (API enabled)
- AO=0 (Native; IMPORTANT!)

## XBee on partner
Establish connection as router in either Transparent (AT) or API mode (AP=0 or 1)
